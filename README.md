# MRP System
設計一個簡單的MRP System，進行整個製造流程的管理 <br>
當收到客戶訂單時，我們需要去檢視產品的原料庫存 <br>
若庫存不足，則須安排叫貨；若庫存足夠，則須安排製造時程，以使訂單能準時交予客戶手中


## 專案介紹
- 呈現方式 : 網頁
- 使用的語法 : php, SQL (Xampp)
- 使用的套件 : Bootstrap
