# A Simple MRP System

## 頁面一 : 物料 list
- 新增、修改
- 欄位
    - 原料名稱
    - Lead time
    - 庫存
      - 數量新增時預設為 0

## 頁面二 : BOM
- 產品結構樹
- 下拉清單顯示詳細資料
- 新增、修改
- 欄位
    - 產品名稱
    - 需要哪些材料、多少數量
    - 排序


## 頁面三 : 排程
- 下拉清單
- 新增、修改
- 欄位
    - Y/M/D 前需要有多少 A、B... 原料
    - 最晚要 Y/M/D 叫 A、B... 原料
    - 狀態
        - 庫存滿足 or 還需要多少數量

## 訂購頁面
- 下拉清單
- 新增、修改
- 欄位
  - 原料名稱
  - 原料總數量
  - 物流狀態
  - 訂購日期
- 下拉顯示詳細訂購資訊
